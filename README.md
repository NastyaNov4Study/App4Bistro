# App4Bistro
# Приложение для заведения "Бисто"
Данное приложение предназначено для просмотра блюд в заведении Бистро
#Структура базы данных
Есть таблицы:
- СоставБлюд
- Продукты
- Блюда
- Рецепты
- Приготовление
# Пользователи
Есть несколько пользователей:
- Администратор
- Диетолог
- Повар
- Бухгалтер
